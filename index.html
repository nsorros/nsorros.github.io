  • 2026
    meeting-recorder macOS · open source

    Local-first macOS meeting recorder. Spots a Meet, Zoom or Teams call, asks before recording, captures the audio with ffmpeg, transcribes on-device with Whisper, then optionally has Claude tidy the raw transcript into notes. Nothing leaves the machine unless you ask.

    macOS Whisper CLI
    01
  • 2026
    menubar macOS · open source

    Two macOS menu-bar widgets, built as xbar plugins — live Claude usage (the 5-hour and weekly windows, straight from the OAuth usage API) and internet speed from a background speedtest probe. Glanceable, with nothing to open.

    macOS Python xbar

  • 2026
    notifly macOS · open source

    A tiny Swift app that posts native macOS notifications carrying your own icon — the thing osascript and terminal-notifier can’t do on current macOS. One build script stamps out a notifier per app; ad-hoc signed, no Apple Developer account.
